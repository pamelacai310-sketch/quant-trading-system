# 欧奈尔CANSLIM交易体系 - 完整使用指南

## 📖 系统概述

本系统完整实现了威廉·欧奈尔（William J. O'Neil）的CANSLIM交易体系及其演化变体，包括：

### 核心模块

1. **CANSLIM 选股器** - 7要素基本面筛选
2. **形态识别器** - 杯柄、双底、VCP等经典形态
3. **口袋支点检测器** - 提前买入信号
4. **策略引擎** - 整合扫描、执行、风险管理
5. **监控熔断** - 实时监控与风险控制

## 🎯 CANSLIM 七要素

### C - Current Earnings (当季收益)
- **标准**：当季EPS增长率≥20-25%
- **权重**：20%
- **实现**：`CANSLIM_Screener._score_c()`

### A - Annual Earnings (年化收益)
- **标准**：年化EPS增长率≥25-30%，连续3年增长
- **权重**：15%
- **实现**：`CANSLIM_Screener._score_a()`

### N - New (新产品/新高)
- **标准**：有新产品催化剂，股价接近52周新高
- **权重**：10%
- **实现**：`CANSLIM_Screener._score_n()`

### S - Supply and Demand (供需关系)
- **标准**：流通盘适中（<10亿股），成交量活跃
- **权重**：10%
- **实现**：`CANSLIM_Screener._score_s()`

### L - Leader (市场领涨股)
- **标准**：相对强度评级(RS)≥70
- **权重**：15%
- **实现**：`CANSLIM_Screener._score_l()`

### I - Institutional Sponsorship (机构资金)
- **标准**：机构持股比例5-70%
- **权重**：15%
- **实现**：`CANSLIM_Screener._score_i()`

### M - Market Direction (大盘趋势)
- **标准**：大盘处于上升趋势
- **权重**：15%
- **实现**：`CANSLIM_Screener._score_m()`

## 📊 经典形态识别

### 1. 杯柄形态 (Cup with Handle)

**特征**：
- 前期上涨至少30%
- 杯底：U形调整，深度15-50%
- 柄部：小幅调整，深度<杯底的1/3
- 枢轴点：柄部顶部
- 成交量：底部收缩，突破放大

**使用**：
```python
from quant_trade_system.patterns import ONeillPatternDetector

detector = ONeillPatternDetector()
patterns = detector.detect_cup_with_handle(price_data)

for pattern in patterns:
    print(f"枢轴点: {pattern.pivot_price}")
    print(f"止损: {pattern.stop_loss_price}")
    print(f"质量: {pattern.quality.value}")
```

### 2. 波动收缩形态 (VCP)

**特征**（Minervini SEPA框架）：
- 3-4次收缩，幅度依次递减（如18% → 12% → 6%）
- 最后一次收缩<10%
- 机构持续吸筹，卖压耗尽
- 突破时成交量放大1.4-1.5倍

**优势**：
- 入场点精准
- 止损可控（7-8%）
- 风险回报比高（常达1:10+）

**使用**：
```python
vcp_patterns = detector.detect_vcp(price_data)

for pattern in vcp_patterns:
    print(f"收缩次数: {len(pattern.vcp_contractions)}")
    print(f"收缩幅度: {[f'{c:.1f}%' for c in pattern.vcp_contractions]}")
```

### 3. 口袋支点 (Pocket Pivot)

**定义**（Gil Morales & Chris Kacher）：
- 股价突破10日均线
- 当日成交量>过去10日所有下跌日成交量
- 表明机构悄悄介入，即将突破

**优势**：
- 买在突破前，成本更低
- 风险更小（止损在10日均线下方）
- 捕捉更多上涨空间

**使用**：
```python
from quant_trade_system.signals import PocketPivotDetector

detector = PocketPivotDetector()
signals = detector.detect_signals(price_data)

for signal in signals:
    print(f"日期: {signal.date}")
    print(f"价格: {signal.price}")
    print(f"成交量倍数: {signal.volume_ratio:.1f}x")
    print(f"止损: {signal.stop_loss_price}")
```

## 📋 交易流程

### 完整决策流程

```
1. 市场环境确认
   ├─ 检查大盘趋势（50日/200日均线）
   ├─ 检查后续交易日（FTD）
   └─ 决策：只在上升趋势时买入

2. CANSLIM 选股
   ├─ 7要素评分
   ├─ 过滤低分股票
   └─ 保留高分股票（≥60分）

3. 形态识别
   ├─ 检测杯柄、VCP等形态
   ├─ 评估形态质量
   └─ 识别枢轴点

4. 信号检测
   ├─ 标准突破信号
   ├─ 口袋支点提前买入
   └─ 成交量确认

5. 风险管理
   ├─ 计算仓位大小（风险1-2%）
   ├─ 设置止损（7-8%）
   ├─ 设置目标（+25%）
   └─ 金字塔加仓

6. 执行与监控
   ├─ 入场交易
   ├─ 实时监控
   ├─ 止损/止盈
   └─ 离场记录
```

## 🎓 最佳实践

### 1. 市场趋势优先

**原则**：永远先确认大盘趋势

```python
# 每日检查
market_analysis = engine.analyze_market(market_data)

if market_analysis['recommendation'] == '等待':
    print("市场环境不佳，等待机会")
    return  # 不进行交易
```

### 2. 严格止损

**原则**：亏损7-8%无条件止损

```python
if position.unrealized_pnl_pct <= -8:
    engine.close_position(
        symbol=symbol,
        exit_price=current_price,
        reason="止损-8%"
    )
```

### 3. 盈利保护

**原则**：获利20-25%部分了结

```python
if position.unrealized_pnl_pct >= 25:
    # 卖出50%
    engine.close_position(
        symbol=symbol,
        exit_price=current_price,
        reason="获利25%部分了结"
    )
```

### 4. 金字塔加仓

**原则**：只在盈利时加仓，从不补仓亏损

```python
if position.unrealized_pnl_pct > 10:
    # 可以加仓
    add_size = original_position * 0.5
else:
    # 绝不加仓
    pass
```
