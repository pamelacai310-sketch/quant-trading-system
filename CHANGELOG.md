## 升级指南

### 从1.0.0升级到1.1.0

1. **更新代码**
```bash
git pull origin main
```

2. **安装新依赖**
```bash
pip install -r requirements.txt
```

3. **重启系统**
```bash
python3 run.py
```

4. **验证升级**
```bash
# 运行测试
python3 -m unittest discover -s tests

# 检查API
curl http://127.0.0.1:8108/api/health
```

### 配置迁移

1.0.0到1.1.0无需配置迁移，所有配置文件向后兼容。

### 数据迁移

数据库schema无变化，无需数据迁移。
